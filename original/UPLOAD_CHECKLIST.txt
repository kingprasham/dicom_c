================================================================================
  TROUBLESHOOTING
================================================================================

Mobile toolbar not showing?
-> Clear browser cache (Ctrl+Shift+Delete)
-> Check mobile-controls.js uploaded
-> Open browser console (F12) for errors

Touch gestures not working?
-> Verify correct tool selected (blue highlight)
-> Touch viewport area, not sidebar
-> Check console for JavaScript errors

Fullscreen not working?
-> iOS/Safari may need user interaction first
-> Try tapping screen before fullscreen
-> Some browsers have limitations

Sync button fails?
-> Must be logged in first
-> Check session timeout (1 hour)
-> Verify API file uploaded
